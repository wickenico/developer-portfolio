Datenschutzerklärung für die Chrome-Erweiterung SummarizeGPT

Stand: Juni 2025

1. Verantwortlicher

Der Entwickler dieser Erweiterung ist allein verantwortlich für den Datenschutz im Rahmen dieser Anwendung. Es erfolgt keine Weitergabe von Nutzerdaten an Dritte.

2. Zweck der Erweiterung

Die Chrome-Erweiterung SummarizeGPT dient ausschließlich dem Zweck, den sichtbaren Text einer vom Nutzer geöffneten Webseite auf Anforderung zusammenzufassen. Die Zusammenfassung erfolgt mithilfe der OpenAI API. Die Erweiterung wird nur auf ausdrückliche Nutzerinteraktion hin aktiv.

3. Welche Daten werden verarbeitet?

  • Webseiteninhalte (Text): Der sichtbare Text der geöffneten Webseite wird lokal durch ein Content Script ausgelesen und zur Zusammenfassung verwendet.
  • OpenAI API-Key: Nutzer können ihren persönlichen OpenAI API-Schlüssel im Popup der Erweiterung eingeben. Dieser wird ausschließlich lokal im Chrome-Speicher (chrome.storage.sync) gespeichert und nicht an den Entwickler oder Dritte weitergegeben. Der Schlüssel wird nur verwendet, um direkt mit der OpenAI API zu kommunizieren.

4. Welche Daten werden nicht erhoben?

SummarizeGPT verarbeitet keine:

  • personenbezogenen Daten (Name, Adresse, E-Mail usw.)
  • Gesundheitsdaten
  • Finanz- oder Zahlungsdaten
  • klassischen Authentifizierungsdaten
  • Kommunikationsinhalte (z. B. E-Mails)
  • IP-Adressen, Standortdaten oder Browserverläufe
  • Nutzerverhalten wie Klicks, Scrolls oder Mausbewegungen

5. Speicherung und Sicherheit

Der OpenAI API-Key wird lokal im Browser des Nutzers gespeichert und kann jederzeit durch den Nutzer gelöscht oder geändert werden. Es findet keine Übertragung oder zentrale Speicherung durch den Entwickler statt.

6. Datenübertragung an Dritte

Die extrahierten Inhalte werden ausschließlich auf Wunsch des Nutzers an die OpenAI API übertragen. Es gelten hierfür zusätzlich die Datenschutzrichtlinien von OpenAI.

7. Ihre Rechte

Da keine personenbezogenen Daten gespeichert oder verarbeitet werden, ist eine Auskunft, Berichtigung oder Löschung nicht erforderlich. Der Nutzer kann jederzeit die Erweiterung deinstallieren oder lokal gespeicherte Daten in Chrome löschen.

8. Kontakt

Bei Fragen zum Datenschutz in dieser Erweiterung kontaktieren Sie bitte den Entwickler über die im Chrome Web Store angegebene E-Mail-Adresse.
